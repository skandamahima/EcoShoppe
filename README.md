# EcoShoppe
Description

EcoShoppe is a full-stack web application designed to promote sustainable living by connecting users with eco-friendly products. The platform provides a seamless shopping experience with features like product search, filtering, and cart management.

 Features:

User authentication (Signup/Login)

Product browsing and search functionality

Product filtering for easy navigation

Add-to-cart system

Responsive design for all devices

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: Node.js, Express.js

Database: MongoDB

 Installation & Setup:

Clone the repository

git clone <your-repo-link>

Navigate to the project folder

cd ecoshoppe

Install dependencies

npm install

Start the server

npm start

Open in browser

http://localhost:3000
 Project Structure:

/frontend → UI components

/backend → Server and API logic

/models → Database schemas

/routes → API routes
